  Claude DevTools へようこそ！
  Getting Started
================================================================================

【このツールは何をするものですか？】

Claude DevToolsは、プログラミングをサポートしてくれる「Claude Code」という
AI※アシスタント※を、より便利に使うためのツールです。

※AI（エーアイ）：人工知能。コンピュータが人間のように考えて作業を手伝ってくれる技術
※アシスタント：助手、補助してくれるプログラム

【何ができますか？】

このツールを使うと、以下のことが簡単にできます：

  ✨ ボタン一つでClaude Codeを起動できる
     → 難しいコマンド※を覚える必要がありません

  ✨ Webブラウザ※と連携して、より高度な作業ができる
     → プログラムの動作確認がスムーズになります

  ✨ 複数のプロジェクト※を簡単に切り替えられる
     → 作業効率が大幅にアップします

※コマンド：コンピュータに指示を出すための文字列
※Webブラウザ：インターネットを見るためのソフト（Edge、Chromeなど）
※プロジェクト：作業を行う場所（フォルダのこと）

【どんな人におすすめですか？】

  👤 Claude Codeを使ってプログラミングを学びたい方
  👤 毎回の設定作業を自動化したい方
  👤 複数のプロジェクトを効率的に管理したい方
  👤 WindowsとLinux※の両方を使っている方

※Linux（リナックス）：Windowsとは別の種類のオペレーティングシステム（OS）

【必要なもの】

このツールを使うには、以下が必要です：

  ■ Windowsパソコン（Windows 10または11）
    → お使いのパソコンがWindowsであることを確認してください

  ■ Linuxサーバー※（リモート※）
    → Claude Codeを実行するための専用のコンピュータ
    → 会社やクラウドサービス※で用意されているものを使います

  ■ インターネット接続
    → WindowsとLinuxサーバーをつなぐために必要です

  ■ Microsoft EdgeまたはGoogle Chrome
    → どちらか一つがあれば大丈夫です

※サーバー：他のコンピュータにサービスを提供する専用のコンピュータ
※リモート：遠隔地にある、離れた場所にある
※クラウドサービス：インターネット上で利用できるコンピュータサービス

【このドキュメントの使い方】

  📄 02_インストール方法.txt
     → 最初にこれを読んで、ツールをセットアップしてください

  📄 03_基本的な使い方.txt
     → セットアップが終わったら、この使い方ガイドを読んでください

  📄 04_よくある質問.txt
     → 疑問があったら、ここを見てください

  📄 05_困ったときは.txt
     → うまく動かないときは、この解決ガイドを参考にしてください

  📄 06_用語集.txt
     → わからない言葉が出てきたら、ここで調べてください

【より詳しく知りたい方は】

  技術的な詳細を知りたい方や、システム管理者※の方は、
  「SystemAdministrator」フォルダ内のドキュメントをご覧ください。

  ※システム管理者：コンピュータシステムを管理・運用する専門家

【サポート】

  わからないことがあったら：
  - まず「06_用語集.txt」で専門用語を確認
  - 「04_よくある質問.txt」で同じ疑問がないか確認
  - 「05_困ったときは.txt」で解決方法を探す
  - それでも解決しない場合は、システム管理者に相談してください

【安全にお使いいただくために】

  ⚠️ このツールは開発作業※専用です
     → 大切なデータは必ずバックアップ※を取ってください

  ⚠️ わからないことは、必ず確認してから実行してください
     → 不明な点があれば、システム管理者に相談しましょう

  ⚠️ パスワードや重要な情報は設定ファイルに書かないでください
     → セキュリティ※のためです

※開発作業：プログラムを作ったり、修正したりする作業
※バックアップ：データのコピーを作って保管すること。万が一の際の保険
※セキュリティ：情報を守るための安全対策
