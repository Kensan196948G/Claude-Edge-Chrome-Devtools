================================================================================
  Claude DevTools インストール方法
  Installation Guide
================================================================================

このガイドでは、Claude DevToolsを使い始めるための準備を、
ステップバイステップ※で説明します。

※ステップバイステップ：一つ一つの手順を順番に進めること

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【事前準備】パソコンの確認
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

まず、お使いのパソコンが以下の条件を満たしているか確認しましょう。

■ステップ1: Windowsバージョン※の確認

  1. キーボードの「Windowsキー」＋「R」を押す
  2. 「winver」と入力して「OK」をクリック
  3. 「Windows 10」または「Windows 11」と表示されればOK

  ※バージョン：ソフトウェアの版、世代のこと

■ステップ2: ブラウザの確認

  以下のどちらかがインストールされていることを確認してください：

  □ Microsoft Edge（エッジ）
    → Windowsに最初から入っているブラウザ
    → タスクバー※に青い「e」のアイコン

  □ Google Chrome（クローム）
    → Googleが提供している無料のブラウザ
    → タスクバーに赤・黄・緑・青の丸いアイコン

  ※タスクバー：画面の一番下にあるバー（細長いエリア）

■ステップ3: インターネット接続の確認

  1. ブラウザを開く
  2. 「https://www.google.com」にアクセスできればOK

【インストール作業】Claude DevToolsのセットアップ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 注意：この作業は、システム管理者と一緒に行うことをおすすめします

■ステップ1: フォルダの準備

  1. エクスプローラー※（フォルダを見るアプリ）を開く
  2. 「D:」ドライブ※を開く
  3. 「Claude-EdgeChromeDevTools」というフォルダがあることを確認

  ※エクスプローラー：ファイルやフォルダを見るためのWindows標準アプリ
  ※ドライブ：パソコンの中のデータ保存場所（C:やD:など）

  もしフォルダがない場合：
    → システム管理者に「フォルダの場所」を確認してください

■ステップ2: Windows Terminalのインストール（推奨）

  Windows Terminalは、より見やすく使いやすいコマンドプロンプト※です。

  ※コマンドプロンプト：文字で指示を出すためのウィンドウ

  【方法1】Microsoft Storeから（簡単）
    1. 「スタートメニュー」を開く
    2. 「Microsoft Store」と検索して開く
    3. 「Windows Terminal」と検索
    4. 「入手」または「インストール」をクリック
    5. インストールが終わるまで待つ（数分）

  【方法2】自動インストール（このツールを使う）
    1. 「Claude-EdgeChromeDevTools」フォルダを開く
    2. 「start.bat」をダブルクリック
    3. メニューで「6」を入力してEnter
    4. 「Y」を入力してEnter
    5. 自動で設定が完了します

  ⚠️ どちらか一つの方法で大丈夫です

■ステップ3: 接続設定の確認

  この部分は **システム管理者に設定してもらう** 必要があります。

  システム管理者に以下を伝えてください：
  「SSH接続※設定をお願いします」

  ※SSH接続：離れたコンピュータに安全につなぐための仕組み

  設定が必要な項目：
  - Linuxサーバーへの接続情報
  - プロジェクトフォルダの場所
  - ポート※番号の設定

  ※ポート：ネットワーク通信の出入り口を識別する番号

■ステップ4: 動作確認テスト

  すべての準備ができたら、正しく動くか確認しましょう。

  1. 「start.bat」をダブルクリック

  2. メニューが表示されます：
     ===============================================
      PowerShell Script Launcher Menu
     ===============================================
     [1] Claude Edge DevTools Setup
     [2] Claude Chrome DevTools Setup
     [3] Edge DevTools Connection Test      ← これを選ぶ
     [4] Chrome DevTools Connection Test
     ===============================================

  3. 「3」と入力してEnterキー※を押す

  4. 以下のような表示が出ればOK：
     ✅ Edge DevTools 接続テスト成功!

  5. Chromeを使う場合は「4」でも同様にテスト

  ※Enterキー：キーボードの右側にある大きなキー。決定する時に使う

■ 「start.bat」が見つからない

  【原因】
    フォルダの場所が違う

  【解決方法】
    1. エクスプローラーで検索ボックスに「start.bat」と入力
    2. 見つかったら、そのフォルダの場所をメモ
    3. システム管理者に場所を確認

■ 「接続テスト失敗」と表示される

  【原因】
    - ブラウザが起動していない
    - 他のプログラムがポートを使っている

  【解決方法】
    1. すべてのブラウザウィンドウ※を閉じる
    2. もう一度テストを実行
    3. それでもダメな場合はシステム管理者に相談

  ※ウィンドウ：画面に表示される四角い枠

■ 「Permission denied」と表示される

  【原因】
    Linuxサーバーへの接続設定が完了していない

  【解決方法】
    システム管理者に「SSH接続設定をお願いします」と依頼

■ エラーメッセージの意味がわからない

  【解決方法】
    1. エラーメッセージをスクリーンショット※で保存
    2. システム管理者に画像を見せて相談

  ※スクリーンショット：画面をそのまま画像として保存すること
    （Windowsキー + Shift + S で撮影できます）

以下の項目すべてにチェックが入れば、インストール完了です！

  □ Windows 10 または 11 を使っている

  □ Windows Terminal がインストールされている（推奨）

  □ システム管理者にSSH接続設定をしてもらった
  □ 接続テスト（メニュー[3]または[4]）が成功した
  □ start.bat をダブルクリックするとメニューが表示される

Q: インストールにどのくらい時間がかかりますか？

A: システム管理者と一緒に行えば、30分〜1時間程度です。
   ただし、SSH接続設定が初めての場合は、もう少しかかる場合があります。

Q: 失敗しても大丈夫ですか？

A: はい、大丈夫です。このツールは既存のデータ※を壊したりしません。
   何度でもやり直せます。

   ※既存のデータ：今あるファイルや設定のこと

Q: 他のパソコンでも使えますか？

A: はい。Windows 10/11であれば、同じ手順でインストールできます。
   ただし、それぞれのパソコンでSSH接続設定が必要です。
