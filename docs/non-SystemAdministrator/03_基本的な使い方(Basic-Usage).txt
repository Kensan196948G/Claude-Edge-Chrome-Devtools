================================================================================
  Claude DevTools 基本的な使い方
  Basic Usage Guide
================================================================================

このガイドでは、Claude DevToolsの日常的な使い方を、
画面の写真のように詳しく説明します。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【起動方法】Claude DevToolsを起動する
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ステップ1: start.batを起動

  1. エクスプローラー※を開く
  2. 「D:\Claude-EdgeChromeDevTools」フォルダを開く
  3. 「start.bat」をダブルクリック

  ※エクスプローラー：ファイルやフォルダを見るアプリ

  💡 ヒント：デスクトップにショートカット※を作ると便利です
     - start.batを右クリック
     - 「ショートカットの作成」を選択
     - 作成されたショートカットをデスクトップに移動

  ※ショートカット：ファイルやフォルダに素早くアクセスするための目印

■ステップ2: メニューから選択

  真っ黒な画面（ターミナル※）に、メニューが表示されます：

  ※ターミナル：文字で操作するウィンドウ

  ===============================================
   PowerShell Script Launcher Menu
  ===============================================

   [Claude DevTools Main]
   1. Claude Edge DevTools Setup
   2. Claude Chrome DevTools Setup

   [Test and Utility]
   3. Edge DevTools Connection Test
   4. Chrome DevTools Connection Test

   [Windows Terminal Settings]
   5. Windows Terminal Setup Guide
   6. Auto-Configure Windows Terminal (PowerShell)

   0. Exit
  ===============================================

  使いたい機能の番号を入力してEnterキー※を押します。

  ※Enterキー：キーボードの右側にある大きなキー

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【基本操作1】Claude Codeを起動する（Edge版）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Microsoft Edgeブラウザ※を使う場合の手順です。

※ブラウザ：インターネットを見るアプリ（EdgeやChromeなど）

■ステップ1: メニューで「1」を選ぶ

  「Enter the number of the script to run:」と表示されたら
  → 「1」と入力してEnter

■ステップ2: ブラウザを選ぶ

  🌐 ブラウザを選択してください:
  [1] Microsoft Edge
  [2] Google Chrome
  番号を入力 (1-2, デフォルト※: 1):

  → 「1」と入力してEnter（Edgeを使う場合）
  → 何も入力せずEnterでもOK（デフォルトがEdgeなので）

  ※デフォルト：初期設定、何も選ばなかった時に自動で選ばれる選択肢

■ステップ3: プロジェクトを選ぶ

  📦 プロジェクトを選択してください

  [1] my-project
  [2] work-project
  [3] test-project

  番号を入力:

  → 作業したいプロジェクト※の番号を入力してEnter

  ※プロジェクト：作業を行う場所、フォルダのこと

■ステップ4: 自動で準備が始まります

  以下のメッセージが順番に表示されます（自動で進みます）：

  ✅ 自動選択されたポート※: 9222
  🌐 Microsoft Edge DevTools 起動中...
  ✅ Microsoft Edge DevTools 接続テスト成功!
  ✅ run-claude.sh 生成完了
  🎨 Claude Code Statusline 設定を生成中...
  🎉 セットアップ完了
  🚀 Claudeを起動します...

  ※ポート：ネットワーク通信の番号のこと（気にしなくてOK）

■ステップ5: Claude Codeが起動します

  しばらく待つと、Claude Codeの画面が表示されます。
  日本語で「こんにちは」などと入力してみましょう！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【基本操作2】Claude Codeを起動する（Chrome版）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Google Chromeブラウザを使う場合は、メニューで「2」を選びます。
それ以外の手順は、Edge版と同じです。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【基本操作3】動作確認テスト
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

問題なく動いているか確認するための機能です。

■Edgeの接続テスト（メニュー[3]）

  メニューで「3」を選ぶと、Edgeブラウザとの接続をテストします。

  成功すると：
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ✅ Edge DevTools 接続テスト成功!
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  📊 接続情報:
     - ポート: 9222
     - Browser: Chrome/120.0.6099.109
     - ...

  失敗すると：
  ❌ Edge DevTools 接続テスト失敗

  → 05_困ったときは.txt を見て対処してください

■Chromeの接続テスト（メニュー[4]）

  メニューで「4」を選ぶと、Chromeブラウザとの接続をテストします。
  表示内容はEdgeと同じです。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【Claude Codeの使い方】起動後の操作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Claude Codeが起動したら、以下のように使います：

■日本語で質問・指示する

  【例1】プログラムについて質問
    「Pythonでファイルを読み込む方法を教えてください」

  【例2】コードのレビュー※を依頼
    「このコードに問題がないかチェックしてください」

  【例3】エラーの解決を依頼
    「このエラーメッセージの意味を教えてください」

  ※レビュー：確認、点検のこと

■画面下部のStatusline※を確認

  Claude Codeの画面の一番下に、このような情報が表示されます：

  📁 /mnt/LinuxHDD/my-project  🌿 main  🤖 sonnet-4.5
  🧠 [████████░░] 80%

  意味：
  📁 今作業しているフォルダ
  🌿 Gitブランチ※（バージョン管理※の名前）
  🤖 使っているClaudeのモデル※
  🧠 メモリ使用率※（どれくらい使っているか）

  ※Statusline：状態表示バー
  ※Gitブランチ：プログラムのバージョン管理で使う名前
  ※バージョン管理：プログラムの変更履歴を記録する仕組み
  ※モデル：AIの種類、性能
  ※メモリ使用率：コンピュータの記憶領域の使用具合

■Claude Codeを終了する

  作業が終わったら：
  1. 「exit」と入力してEnter
  2. または「Ctrl + C」キー※を押す

  ※Ctrl + C：キーボードの「Ctrl」キーを押しながら「C」キーを押す

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【便利な機能】プロジェクトの切り替え
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

複数のプロジェクトで作業する場合：

  1. Claude Codeを終了（exit）
  2. start.batを再度起動
  3. メニューで「1」または「2」を選ぶ
  4. 別のプロジェクト番号を選ぶ

  💡 ヒント：プロジェクトごとに別々のウィンドウで同時に起動できます
     （異なるポート番号が自動で割り当てられます）

【よくある操作】チートシート※
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

※チートシート：よく使う操作をまとめた早見表

┌─────────────────────────────────────────────────────┐
│ やりたいこと               │ 操作                       │
├─────────────────────────────────────────────────────┤
│ Claude Codeを起動          │ start.bat → [1]または[2]   │
│ プロジェクトを切り替え     │ exit → start.bat → 別番号  │
│ 接続確認                   │ start.bat → [3]または[4]   │
│ Claude Codeを終了          │ exit または Ctrl+C         │
│ ウィンドウを閉じる         │ 右上の×ボタン              │
│ メニューに戻る             │ 0 → Enter                  │
└─────────────────────────────────────────────────────┘

【注意事項】安全に使うために
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■途中で画面が止まったら

  ⚠️ むやみにウィンドウを閉じないでください

  対処方法：
  1. 5分ほど待ってみる（処理中の場合があります）
  2. それでも動かない場合は「Ctrl + C」
  3. それでもダメなら、システム管理者に相談

■エラーメッセージが出たら

  ⚠️ パニックにならず、落ち着いて対処しましょう

  対処方法：
  1. メッセージをスクリーンショット※で保存
  2. 「05_困ったときは.txt」を確認
  3. 解決しない場合はシステム管理者に相談

  ※スクリーンショット：Windowsキー + Shift + S で画面を撮影

■複数のウィンドウが開いたら

  正常です。以下のウィンドウが開きます：

  □ ターミナル（黒い画面）← メインウィンドウ
  □ ブラウザ（EdgeまたはChrome）← 自動起動
  □ Claude Code（AIとの対話画面）← Linux側で起動

  すべて連携して動いています。
