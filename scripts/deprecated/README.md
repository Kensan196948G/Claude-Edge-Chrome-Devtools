# 非推奨スクリプト (Deprecated)

このディレクトリには v1.3.0 以前に使用されていた個別ブラウザスクリプトが含まれます。

## 代替手段

統合スクリプト `scripts/main/Claude-DevTools.ps1` を使用してください:

```powershell
# Edge版
.\scripts\main\Claude-DevTools.ps1 -Browser edge

# Chrome版
.\scripts\main\Claude-DevTools.ps1 -Browser chrome
```

## 含まれるスクリプト

- `Claude-EdgeDevTools.ps1` — 旧Edge専用スクリプト (v1.3.0で非推奨)
- `Claude-ChromeDevTools-Final.ps1` — 旧Chrome専用スクリプト (v1.3.0で非推奨)

これらのスクリプトは参照用に保持されていますが、今後のメンテナンスは行われません。
